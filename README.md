FamilyBilling
================

本项目缘于一个AngularJS初学者想要将其应用于实践的想法，为家庭提供一个取代记事本和电子表格的财务管理解决方案。毕竟，任何技术总是要用于解决实际问题的。

技术选型：AngularJS + SpringMVC + Hibernate; Bootstrap, Restful API, Maven。

###开发工具及部署环境
JDK v1.8
Maven 3.2.3     
Git 1.8.4     
MySQL 5.5.11        
Tomcat 8.0.3  

Spring 4.1.1<br/>
Hibernate 4.3.6<br/>
fasterxml.jackson 2.1.1<br/>

首先须导入familybilling.sql

###已完成的功能
费用分类管理<br/>
开支管理<br/>

###AngularJS重点
双向数据绑定<br/>
ngRoute - 多重视图和路由<br/>
ngResource - 通过REST API与服务器交互<br/>
Filter
